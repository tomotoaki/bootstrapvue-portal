# ポータル

## Requirement

- Vue.js v2.6.12

- BootstrapVue 2.21.2

## Installation

※VSCodeのインストール手順が必要であれば追加する。

## Usage

1. Start

    ```
    npm start
    ```
    または
    ```
    npx http-server -a localhost -p 8000 -o 
    ```

    サーバ起動後、以下 URL にアクセスする。

    [http://localhost:8000/](http://localhost:8000/)

3. Stop

    ```
    Ctrl + C
    ```
